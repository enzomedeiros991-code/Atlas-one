# Atlas One

Projeto inicial de um planetário digital com a Terra centralizada em 3D.

## O que já tem

- Terra 3D centralizada
- Nuvens
- Atmosfera
- Sol
- Lua
- Estrelas
- Constelações estilizadas
- Horário local e UTC em tempo real
- Fase da Lua aproximada
- Painel futurista

## Como rodar

1. Extraia o ZIP.
2. Abra a pasta no VS Code ou editor.
3. Use uma extensão tipo Live Server.
4. Abra `index.html`.

Importante: por usar módulos JavaScript e Three.js, pode não abrir certinho só clicando direto no arquivo. O ideal é rodar em servidor local.
